# ba_hotel_reservation

Application Flutter pour la réservation d'hôtels (prototype).

Description
-----------
Ce dépôt contient une application mobile Flutter destinée à gérer la réservation d'hébergements : affichage d'hôtels, détails, réservation et pages administratives de base.

Principales fonctionnalités
- Parcours des établissements (liste, filtres simples)
- Fiche détaillée d'un hôtel
- Processus de réservation basique
- Structure multiplateforme (Android / iOS)

Prérequis
- `Flutter` (version stable récente)
- SDK Android / Xcode si vous ciblez les plateformes natives

Installation et exécution
1. Installez les dépendances :

```
flutter pub get
```

2. Lancez l'application sur un émulateur ou appareil connecté :

```
flutter run
```

Structure du projet
- `lib/` : code Flutter (point d'entrée `main.dart`)
- `android/`, `ios/` : configurations natives
- `test/` : tests unitaires/widget
